Tags

Here is to show what kinds of tags in HTML using table. It has Tage Name, Uses, Explain

Tag Name Description
head It is used to gather metadata for this web pages like title, style, link, meta, etc..
body It decides the area of contents such as image, paragraphs, etc. There is only one body tag.
html It dictates that this is a HTML document. It is a root tag of any others.
title It represent the title of link on the browsers tap.
meta It is a tag for metadata of HTML document. It can represent many meta information.
div It is divide HTML layout.
a It makes hyper link text.
script This tag dictates a script to be run.
link It is used to make relationships with other files such as CSS.
img It is used to upload image on to the web page with other tags that describe the information to show image.
p It is a tags for paragraphs. It devides each paragraphs.
span It is in-line element to combine some part of text.
li It is a tag for list item. It should be in list tags.
ul It is a tag for unordered list.
br It is a tag to make new line.
style It defines document style information. It usually dictates what CSS is to be applied.
h1 It is used to represent the most important head.
h2 It is a tag for the head that secondary.
input It represent input field from user.
form It represent a HTML form so that user can make inputs from web pages.
strong It is a bold tag.
h3 It is the third head tag.
table It makes table form. And it is used with "tr" or "td" tags.
tr It is table row tag.
td It is table column tag.
